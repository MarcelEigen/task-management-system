# Task Management System

... English description ...

A simple fullstack task and project management application built with:

- Spring Boot (Backend)
- H2 Database
- HTML/JavaScript (Frontend)
- REST API

The project allows users to:
- Create and manage projects
- Add, update, and delete tasks
- Assign tasks to projects
- Mark tasks as completed

This project demonstrates fullstack development skills and REST architecture.

---

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- HTML / JavaScript
- REST API

---

## 🚀 Setup

### Prerequisites

- Java 17+ installed
- Maven

### Run Application

```bash
mvn spring-boot:run
```

Application runs at: http://localhost:8080
